# 관제 대시보드

## 1. 개요
- **화면명**: 관제 대시보드
- **목적**: 실시간 및 주간 세차 운영 현황을 모니터링하고, 전주 대비 추이 분석 및 파트너별 수행 역량을 시각화하여 운영 리스크에 대응합니다.
- **주요 기능**: 당일 세차 수행 흐름 파악, 긴급 이슈 처리 지표 확인, 전주 대비 오더 추이 분석, 파트너별 적시 수행 현황 모니터링.

## 2. KPI 현황

### 2.1 세차 수행 현황 (Today)
당일 발생한 전체 오더의 실시간 라이프사이클 흐름을 보여줍니다.

- **구성**:
  - **발행**: 당일 전체 오더 발행 건수.
  - **예약 완료**: 수행원이 배정되어 예약된 건수.
  - **수행 완료**: 세차가 최종 완료된 건수.
  - **취소**: 발행 후 취소된 오더 건수.

### 2.2 긴급 세차 처리 현황 (최근 1주일)
위생 장애 및 긴급 이슈에 대한 대응 성과를 모니터링합니다.

- **지표 항목**:
  - **긴급 세차 오더**: 이슈 대응을 위해 발행된 긴급 오더 건수. (소스: 위생 장애 또는 ML 판단)
  - **위생 장애 인입**: 접수된 위생 장애 티켓 건수.
  - **적시 수행 (72시간 이내)**: 오더 발행 후 **72시간(3일) 이내**에 완료된 건수.
  - **평균 장애 처리 리드타임**: 이슈 인입부터 처리 완료까지 소요된 평균 시간 (**시간 단위**로 표시).

## 3. 데이터 시각화

### 3.1 전주 대비 일자별 오더
요일별 오더 발생량을 전주 데이터와 비교하여 주간 트렌드를 파악합니다.

- **차트 유형**: Composed Chart (Area + Bar)
- **시각화 로직**:
  - **전주 (배경)**: 연한 회색(#F1F5F9)의 면적 차트(Area)로 배치하여 기저 추이로 표현.
  - **금주 (오버레이)**: Royal Blue(#0052CC) 색상의 막대 그래프(Bar)를 전주 데이터 위에 겹쳐서 현재 도달 수준을 직관적으로 비교.

### 3.2 파트너별 수행 현황 (최근 1주일)
각 파트너사의 오더 처리 능력 및 지연 리스크를 72시간 기준으로 분석합니다.

- **차트 유형**: 세로형 막대 그룹 차트 (Grouped Bar Chart)
- **데이터 구성**: 파트너별로 아래 3개 요소를 나란히 표시하여 비교.
  - **적시수행 (Royal Blue)**: 72시간 이내 완료 건.
  - **미수행 (Light Gray)**: 72시간 미경과 상태의 미완료 건.
  - **지연 (Rose Red)**: 72시간을 초과한 미완료 건.

### 3.3 시간대별 세차 집중도 (최근 1주일)
세차 시작 시간 기준의 작업 밀집도를 시각화하여 피크 타임을 분석합니다.

- **차트 유형**: 영역형 차트 (Area Chart)
- **기준**: 실제 세차 시작 시간.
- **스타일**: 그라데이션 채우기 효과 적용.

### 3.4 장애 처리율 (최근 1주일)
전체 장애 발생 건수 대비 해결 완료된 비중을 보여줍니다.

- **차트 유형**: 도넛 차트 (Pie Chart)
- **특징**: 차트 중앙에 전체 완료율(%) 수치 강조.

## 4. 비즈니스 로직 및 기준
| 구분 | 정의 및 로직 |
| :--- | :--- |
| **적시수행 기준** | 오더 발행 시점부터 **72시간(3일) 이내**에 '완료' 상태로 전환된 경우. |
| **지연 기준** | 오더 발행 후 **72시간을 초과**했음에도 여전히 '완료'되지 않은 경우. |
| **장애 처리 리드타임** | 장애 인입 시점부터 완료 시점까지의 총 소요 시간을 **시간(Hour) 단위**로 집계하여 평균 산출. |
| **전주 대비 추이 (WoW)** | 전주 동일 요일의 데이터와 금주 데이터를 동일 좌표계에 배치하여 실적 증감 파악. |
