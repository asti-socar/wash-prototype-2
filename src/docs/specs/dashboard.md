# 관제 대시보드

## 1. 개요
- **화면명**: 관제 대시보드
- **목적**: 실시간 세차 운영 현황을 모니터링하고, 위생 장애 등 리스크 상황에 신속하게 대응하기 위한 종합 지표를 제공합니다.
- **주요 기능**: 당일 세차 수행 흐름 파악, 긴급 이슈 처리 현황 확인, 주요 운영 지표 시각화.

## 2. KPI 현황

### 2.1 세차 수행 현황
당일 발생한 오더의 전체 라이프사이클 흐름을 보여줍니다.

- **데이터 소스**: 확인 가능
- **구성**:
  1. **발행**: 전체 오더 발행 건수.
  2. **예약 완료**: 수행원이 배정되어 예약된 건수.
  3. **수행 완료**: 세차가 완료된 건수.
  4. **취소**: 취소된 오더 건수.
- **시각화**: 각 단계별 수치를 나열하고, 단계 사이에 구분선을 두어 흐름을 표현합니다.

### 2.2 긴급 세차 처리 현황 (최근 1주일)
위생 장애 및 긴급 이슈에 대한 대응 성과를 모니터링합니다.

- **데이터 소스**: 확인 가능
- **지표 항목**:
  - **위생 장애 인입**: 접수된 위생 관련 이슈 건수.
  - **긴급 오더 발행**: 이슈 대응을 위해 발행된 긴급 오더 건수.
  - **적시 수행**: 서비스 수준 협약 내에 처리된 건수.
  - **평균 리드타임**: 이슈 접수부터 처리 완료까지 소요된 평균 시간.

## 3. 데이터 시각화

### 3.1 일자별 오더
요일별 오더 발생량을 비교하여 주간 트렌드를 파악합니다.

- **차트 유형**: 세로형 막대 그래프
- **데이터 소스**: 확인 가능
- **축 정보**:
  - **X축**: 요일
  - **Y축**: 오더 건수
- **스타일**: Royal Blue 색상의 막대 그래프.

### 3.2 파트너별 수행률
각 파트너명의 세차 수행 달성률을 비교합니다.

- **차트 유형**: 가로형 막대 그래프
- **데이터 소스**: 확인 가능
- **축 정보**:
  - **Y축**: 파트너명
  - **Bar**: 수행률
- **특징**: 막대 우측에 수행률 수치(%)를 직접 표시하여 가독성을 높임.

### 3.3 시간대별 세차 집중도
하루 중 세차 작업이 집중되는 시간대를 시각화합니다.

- **차트 유형**: 영역형 차트
- **데이터 소스**: 확인 가능
- **축 정보**:
  - **X축**: 시간
  - **Y축**: 집중도 값
- **스타일**: 그라데이션 채우기 효과를 적용하여 밀도 표현.

### 3.4 장애 처리율
전체 발생한 장애 건수 대비 처리 완료된 비율을 보여줍니다.

- **차트 유형**: 도넛 차트
- **데이터 소스**: 확인 가능
- **구성**:
  - **완료**: 처리된 건수 비중 (Royal Blue)
  - **미처리**: 대기 중인 건수 비중 (Light Gray)
- **특징**: 차트 중앙에 전체 완료율(%)을 텍스트로 강조 표시.
