# 파트너 담당자 조회

## 1. 개요
- **목적**: 파트너사에 소속된 인력(관리자/수행원) 규모와 조직 구조를 파악하고 공백을 방지합니다.

## 2. 주요 기능
### 2.1 담당자 리스트
- **필터**: 소속 파트너사 선택, 계정 유형(관리자/수행원), 계정 상태.
- **데이터 그리드**: 성명, 소속 파트너사, 유형, 배정된 쏘카존 현황(개수), 마지막 접속일.

### 2.2 담당자 상세 (Drawer)
- **상세 프로필**: 계정 상태(활성/정지) 확인 및 프로필 정보.
- **쏘카존 목록**: 해당 인력에게 배정된 모든 담당 쏘카존 목록을 리스트로 노출.
