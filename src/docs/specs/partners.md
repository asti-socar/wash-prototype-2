# 파트너 관리

## 1. 개요
- **목적**: 계약된 파트너사의 상태와 담당 구역을 설정하고 세차 수행 현황을 체계적으로 관리합니다.

## 2. 주요 기능
### 2.1 파트너 리스트
- **필터**: 파트너 상태(운영중/중지/종료), 담당 지역1/2, 파트너 명 검색.
- **데이터 그리드**: 파트너 ID, 파트너 명, 상태 배지, 담당 지역, 연락처, 등록일.
- **특이사항**: 과거 계약이 종료된 파트너사도 필터를 통해 노출 가능해야 합니다.

### 2.2 파트너 상세 및 관리 (Drawer)
- **기본 정보**: 상세 연락처, 계약 조건, 수행 가능 범위 조회.
- **지역 할당**: 파트너사가 관리할 복수 개의 지역(Region/Zone ID)을 선택 및 매핑.
- **등록/수정**: 신규 파트너 등록 및 상태값(운영중/중지/종료) 변경 기능.
