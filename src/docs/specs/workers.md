# 수행원 조회

## 1. 개요
- **목적**: 현장 수행원의 배치 현황을 모니터링하고 품질 이슈 발생 시 피드백 근거로 활용합니다.

## 2. 주요 기능
### 2.1 수행원 리스트 (View Only)
- **필터**: 지역별 조회, 소속 파트너사별 조회.
- **데이터 그리드**: 수행원 성명, 소속 파트너사, 배정된 쏘카존 명, 벌점/수행 점수.

### 2.2 수행원 상세 (Drawer)
- **현황 파악**: 어느 회사 소속인지, 어떤 쏘카존들에 배정되어 있는지 상세 확인.
- **정책**: 인터널 어드민에서는 모니터링 목적의 '뷰잉'만 제공하며, 편집은 생략합니다.
