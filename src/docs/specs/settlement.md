# 합의 요청 관리

## 1. 개요
- **화면명**: 합의 요청 관리
- **목적**: 세차 수행 중 발견된 특이사항(심각한 오염 등)으로 인한 추가 비용 발생 시, 파트너사와 운영자 간의 합의 프로세스를 관리합니다.
- **주요 기능**: 합의 요청 목록 조회, 현장 사진 및 사유 확인, 청구 금액 조정(네고), 승인 및 반려 처리.

## 2. 기능 명세

### 2.1 합의 요청 목록
파트너사로부터 접수된 추가 비용 합의 요청 건들을 조회합니다.

- **데이터 소스**: 확인 가능
- **행 식별자**: 확인 가능

#### 컬럼 정의
| 헤더명 | 비고 |
| :--- | :--- |
| **오더 ID** | 원본 오더 식별자 |
| **차량번호** | |
| **차종** | |
| **쏘카존** | |
| **파트너사** | 요청한 협력사 |
| **요청 시간** | |
| **상태** | 요청, 수락, 거절, 네고 등 상태에 따른 배지 표시 |

### 2.2 합의 요청 상세
목록 클릭 시 우측 Drawer를 통해 상세 정보를 확인하고 의사결정을 수행합니다.

#### 주요 섹션 및 필드
1. **요청 정보**
   - **기본 정보**: 오더 ID, 차량 정보, 파트너사.
   - **요청 사유**: 예: 오염도 심각, 특수 오염 등.
   - **세차 항목**: 추가 수행이 필요한 항목 리스트.
   - **청구 금액**: Input 필드
     - 상태가 '요청'일 경우 수정 가능 (네고 가능).
   - **코멘트**: 파트너사 코멘트 또는 처리 결과 메시지.

2. **현장 사진**
   - 세차 전 상태를 증빙하는 사진을 확인합니다. (현재 프로토타입은 예시 이미지 제공)

#### 처리 동작
상태가 **'요청'**일 경우에만 활성화됩니다.

- **승인**: '승인' 버튼 클릭 시 상태가 '수락'으로 변경되며, 입력된 청구 금액으로 확정됩니다.
- **반려**: '반려' 버튼 클릭 시 사유 입력창이 활성화됩니다.
  - **반려 확정**: 반려 사유를 입력하고 확정하면 상태가 '거절'로 변경됩니다.
  - **취소**: 반려 입력을 취소하고 상세 화면을 유지합니다.
