# 차량 관리

## 1. 개요
- **화면명**: 차량 관리
- **목적**: 전체 차량의 세차 현황 모니터링, 장기 미세차 차량 식별 및 존별 차량 분포 확인.
- **주요 기능**: 차량 목록 조회, 다차원 필터링, 차량 상세 정보 및 세차 이력 조회.

## 2. 기능 명세

### 2.1 검색 및 필터
상단 카드 영역에 위치하며, 리스트 데이터를 실시간으로 필터링합니다.

| 구분 | 필드명 | 타입 | 설명 및 로직 |
| :--- | :--- | :--- | :--- |
| **검색** | 통합 검색 | Input | **대상 필드 (OR 조건)**:<br>- 차량번호<br>- 존 이름<br>- 존 ID<br>*대소문자 구분 없음* |
| **필터** | 지역1 | Select | 지역1 필드 기준 필터링. 변경 시 지역2 필터 초기화. |
| **필터** | 지역2 | Select | 지역2 필드 기준 필터링. 선택된 지역1에 속한 지역2 목록만 노출. |
| **필터** | 파트너명 | Select | 파트너명 필드 기준 필터링. |
| **초기화** | 필터 초기화 | Button | 모든 검색어 및 필터 조건을 초기화합니다. |

### 2.2 차량 목록
필터링된 결과를 테이블 형태로 표시합니다.

- **데이터 소스**: 확인 가능
- **행 식별자**: 확인 가능
- **정렬**: 원본 데이터 순서 유지

#### 컬럼 정의
| 헤더명 | 비고 |
| :--- | :--- |
| **차량번호** | |
| **존이름** | |
| **존 ID** | |
| **지역1** | |
| **지역2** | |
| **파트너명** | |
| **진행중 오더** | 값이 있을 경우 버튼 형태로 노출 (클릭 시 오더 상세 이동 가능성 암시), 없을 경우 `-` 표시 |
| **마지막 세차일** | YYYY-MM-DD 형식 |
| **세차 경과일** | 마지막 세차일 기준 현재 날짜와의 차이 계산 |

### 2.3 차량 상세
목록의 행을 클릭하면 우측에서 슬라이딩되는 Drawer입니다.

- **진입 트리거**: 테이블 행 클릭
- **타이틀**: 차량 상세 - {차량번호}

#### 섹션 구성
1. **기본 정보**
   - **차량번호**
   - **차종**
   - **연료유형**
   - **존이름**: {존이름} ({존 ID})
   - **지역1**: {지역1}
   - **지역2**: {지역2}
   - **파트너명**
   - **진행중 오더**: (없을 시 "없음")
   - **마지막 세차일**
   - **세차 경과일**: {세차경과일}일

2. **세차 이력**
   - 현재 프로토타입 단계에서는 예시 데이터 리스트를 표시함.
   - 예: 2026-01-10, 라이트 세차, 완료 등

## 3. 데이터 로직 참고
- **경과일 계산**: 유틸리티 함수를 사용하여 (오늘 - 마지막 세차일) 로직으로 일수 계산.
- **필터링**: 검색어와 필터 상태가 변경될 때마다 결과 리스트 재계산.
