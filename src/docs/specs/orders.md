# 오더 관리 (Order Management)

## 1. 개요
- **화면명**: 오더 관리 (OrdersPage)
- **목적**: 세차 오더의 전체 생명주기 관리(발행, 배정, 수행, 완료) 및 특수 미션/이슈 관제.
- **주요 기능**: 오더 목록 조회/필터링, 수기 오더 발행, 미션 등록, 상세 정보(타임라인, 사진, 점검) 확인.

## 2. 기능 명세

### 2.3 오더 발행 및 미션 (Actions)

#### 수기 오더 발행 (Manual Creation)
- **트리거**: '오더 발행' 버튼
- **입력 필드**:
  - 차량번호 (`plate`) *필수
  - 차종 (`model`)
  - 존 이름 (`zone`) *필수
  - 세차 타입 (`washType`)
- **생성 로직**:
  - `orderGroup`: "수시", `orderType`: "수시세차", `partnerType`: "현장"으로 고정 생성.
  - **미션 연동**: 입력된 `plate`에 해당하는 `pending` 상태의 미션이 있을 경우, 자동으로 오더의 `attachedMissions`에 포함됨.

#### 미션 등록 (Mission Registration)
- **트리거**: '미션 등록' 버튼
- **입력 필드**: 대상 차량번호(`plate`), 미션 내용(`content`)
- **로직**:
  - 신규 미션은 `status: "pending"`으로 등록됨.
  - 오더 수행 완료 시, 해당 오더에 포함된 미션들의 상태가 `completed`로 변경되어 다음 오더 생성 시 제외됨.

### 2.4 오더 상세 (Detail Drawer)
오더 클릭 시 우측 Drawer에서 탭(Tab) 구조로 상세 정보를 제공합니다.

#### 탭 구성
1. **상세정보 (`info`)**
   - **기본 정보**: 오더 ID, 오더 구분/유형, 차량 정보, 존, 파트너 유형, 수행원, 상태, 메모 등.
   - **금액/연계**: 최종 청구 금액, 연계 오더(Parent Order) 링크.

2. **사진 및 이력 (`history`)**
   - **진행 이력 (Timeline)**: `partnerType`에 따라 다른 단계 노출.
     - **입고**: 미배정 -> 파트너 배정 -> 예약 -> 입고 중 -> 수행 중 -> 세차 완료 -> 출고 중 -> 완료
     - **현장**: 미배정 -> 예약 -> 수행 중 -> 완료
   - **수행 사진**: 수행 전/후 사진 그리드 (클릭 시 확대).
   - **점검 리스트**: 외관, 휠/타이어, 실내, 쓰레기 수거 등 Pass/Fail 체크.

3. **미션 및 분실물 (`mission`)**
   - **미션**: 오더에 할당된 특별 미션 목록 및 수행 상태.
   - **분실물**: 습득된 분실물 정보 (습득물 명, 보관 상태, 습득 장소).
